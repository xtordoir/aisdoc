About Alma in Silico APIs

This set of APIs and tools is developed by the alma in Silico bioinformatics team.

With a cloudy approach, we aim to provide a platform to manage data storage, actions and annotations on these data, laboratory services and data processing services.

Providers can plug their resources into the system and expose their services to granted users. A support for resources usage logging for billing purposes is also offered.

Each API provided by AIS is a web-based service, any operation is accessible through HTTP requests via a REST interface. The HTTP requests are wrapped in Command Line tools, allowing an easy programmatic access to the service stack.

There are 4 categories in the service stack:

  • Services
  • Storage
  • Application
  • Access and Billing
